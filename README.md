# BizMath
BizMath — калькулятор амортизации и ликвидности.

Автоматизирует расчёты трёх методов амортизации и трёх коэффициентов ликвидности для малого бизнеса.

Главная – описание, карточки-ссылки.
Амортизация – форма (стоимость, срок, метод) → таблица + график.
Ликвидность – форма (активы, обязательства, запасы, деньги) → три коэффициента с цветовой индикацией.
Список активов – таблица сохранённых объектов, редактирование/удаление.
